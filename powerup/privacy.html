trelly privacy

The Power-Up

The trelly Power-Up runs entirely in your browser using Trello's Power-Up client API. It reads only the card data Trello hands it (name, URL, badges, attachments) to render its sections and copy buttons.

  • No authorization or token is requested — it cannot read anything you can't already see.
  • No requests to any server other than Trello's CDN (which serves the Power-Up client library).
  • Nothing is stored, logged, or transmitted. No analytics, no cookies.

The CLI and MCP server

trelly and trelly-mcp (installed separately via npm or Homebrew) run locally on your machine. Credentials live only in ~/.config/trelly/config.json (file mode 600). API requests go directly to api.trello.com; the Atlassian privacy policy applies to that traffic. The package collects no analytics, crash reports, or usage metrics.
